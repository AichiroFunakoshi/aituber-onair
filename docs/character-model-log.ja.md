# Character Model Log

この文書は、初期PNGTuberキャラクター画像を作る過程で決めた方針と、
生成時に避けるべき傾向を記録するものです。目的は、今後の画像生成や
4状態PNG作成で、同じ判断を繰り返さずに安定した方向へ寄せることです。

## 目的

`apps/tuber` の初期キャラクターとして使う、バストアップ正面向きの
PNGTuber用イラストを作成します。

最終的には以下の4状態が必要です。

- `mouth_close_eyes_open.png`
- `mouth_close_eyes_close.png`
- `mouth_open_eyes_open.png`
- `mouth_open_eyes_close.png`

背景は透過にします。衣服はキャラクター性を壊さない範囲で簡略化します。

## 参照した人物特徴

参照写真から、以下の要素を優先します。

- 黒から暗いワインブラウンの短い丸みのあるボブ。
- 目に近い重めの前髪。中央付近に不均一な割れがある。
- 頬の横にかかるサイドヘア。
- 透明感のある明るい肌。
- 柔らかい頬。
- 小さめで丸い顎。尖らせない。
- 小さく整った鼻。
- 控えめな赤みのある小さな唇。
- 落ち着いた正面視線。
- 暗いグレーからブラウン寄りの目。青、紫、発光表現にはしない。
- やや横長のアーモンド型の目。大きすぎる丸目にはしない。
- 目元の小さな泣きぼくろ。
- 下唇付近のピアスは、必要に応じて省略または簡略化する。

## これまでの試行で分かったこと

正面化、背景削除、衣服変更、デフォルメを同時に強めると、
顔が一般的なアニメ顔に寄り、参照人物らしさが薄くなります。

特に以下の傾向は避けます。

- 目を青や紫にする。
- 目を大きく丸くしすぎる。
- 顎を尖らせてクールさを出そうとする。
- 髪の線画を細かく入れすぎる。
- 髪の量を増やしすぎて顔より髪が主役になる。
- 服のレース、透け、細かい模様を描き込みすぎる。
- 幼く、かわいらしい方向へ寄せすぎる。
- 大人っぽさを出すために首を長くしすぎる。

「クールにする」と「顎を尖らせる」は別です。このキャラクターでは、
丸みのある顎と柔らかい頬を残したまま、目元と表情の落ち着きで
クールさを出します。

## 現在の採用方針

現時点では、以下の方向を基準にします。

- バストアップ、正面向き。
- 顔を主役にする。
- 髪のボリュームは控えめにし、顔との比重を整える。
- 前髪は重めに残すが、目元が隠れすぎないようにする。
- 髪の内側の線は少なめにする。
- 目は暗め、横長、落ち着いた印象にする。
- 顎は小さく丸く、尖らせない。
- 頬は柔らかく残す。
- 泣きぼくろを入れる。
- 服はシンプルな黒系トップスまたは簡略化した襟元にする。
- 背景は透過。

## 次回生成時の指示メモ

次に生成する場合は、以下の方向で調整します。

- 髪の内側の線をさらに減らす。
- 顎を少し丸くする。
- 首を少し短くする。
- 頬の柔らかさを戻す。
- 顔の面積を確保し、髪が主役にならないようにする。
- 目の色を暗いグレーまたはブラウン寄りに固定する。
- 幼くしすぎず、ただし大人っぽくしすぎない。

## PNGTuber 4状態化の注意

4状態は同じベース絵から作ります。状態ごとに顔の比率や髪型が変わると、
口パクやまばたき時に別人感が出るためです。

- 目開きと目閉じでは、顔の輪郭、髪、服、首の位置を変えない。
- 口閉じと口開きでは、唇と口の開きだけを変える。
- 口開き時も表情を大きく変えすぎない。
- 透過背景とキャンバスサイズを全状態で揃える。
- ファイル名はアプリ側の想定名と一致させる。

## 採用前の確認観点

アプリに組み込む前に、以下を確認します。

- 小さく表示しても顔と目元が読める。
- 目の色が参照人物の印象から外れていない。
- 髪が重すぎず、顔との比重が自然。
- 顎が尖りすぎていない。
- 泣きぼくろが残っている。
- 服の情報量が多すぎない。
- 4状態を切り替えても別人に見えない。
